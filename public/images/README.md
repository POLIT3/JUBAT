## Image Guidelines

- **Format:** JPG, PNG, or WebP
- **Size:** 800-1200px width for optimal quality
- **Quality:** High resolution (72-96 DPI for web)
- **Alt text:** Already configured in HTML

## Quick Setup

1. Create images with the exact filenames listed above
2. Place them in this `images/` directory
3. Use real photos or high-quality stock images
4. Ensure images match the water/purity theme
